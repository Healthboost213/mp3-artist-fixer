# Bulk MP3 Artist Fixer
Some downloaders tend to include all the artists that worked on a track in the tags. Some music players don't play well with them, causing issues such as albums being split, artists not categorized seperately and more when browsing. I made a script that fixes the metadata (artist, albumartist) to only keep the primary artist who is usually the one that made the album.

The folder structure is a little rigid but you can change it easily by modifying source and adjusting the iterations. The path needs to be changed before running as well. It's `'.\music'` by default.

**Assumed folder structure:**  
Music Folder > Artist Name > Album Name > Track Name
